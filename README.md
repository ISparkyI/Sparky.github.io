# Портфоліо

Готовий сайт-портфоліо: `index.html`, `style.css`, `script.js`. Ніяких збірок чи залежностей — просто редагуйте файли вручну.

## Що змінити перед публікацією

1. **`index.html`** — замініть:
   - `Ім'я Прізвище` на своє ім'я (зустрічається кілька разів)
   - посилання `github.com/your-username` та `linkedin.com/in/your-username`
   - `your.email@example.com` на свою пошту
   - текст у секціях about, projects, skills, contact на свій
   - `<title>` та `<meta name="description">` у `<head>`

2. **`style.css`** — за бажанням поміняйте кольори у блоці `:root` на початку файлу (акцентні кольори `--accent-*`).

## Як опублікувати на GitHub Pages

1. Створіть новий репозиторій на GitHub. Якщо хочете, щоб сайт був доступний за адресою `https://ваш-нік.github.io`, назвіть репозиторій саме так: `ваш-нік.github.io`. Для будь-якої іншої назви сайт буде доступний за адресою `https://ваш-нік.github.io/назва-репозиторію`.
2. Завантажте туди `index.html`, `style.css`, `script.js` (через веб-інтерфейс GitHub — кнопка "Add file → Upload files", або через git).
3. У репозиторії відкрийте **Settings → Pages**.
4. У розділі "Build and deployment" виберіть **Source: Deploy from a branch**, гілку `main` і папку `/root`, натисніть **Save**.
5. Через 1–2 хвилини сайт зʼявиться за посиланням, яке GitHub покаже на цій же сторінці.

## Локальний перегляд

Просто відкрийте `index.html` у браузері — жоден сервер не потрібен.
