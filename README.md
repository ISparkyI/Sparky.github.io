# Портфоліо проєктів

Багатосторінковий сайт-портфоліо без збірки — просто HTML/CSS/JS. Дві сторінки:

- **index.html** — головна: hero + проєкти, розділені на категорії з фільтром
- **contact.html** — форма для звʼязку щодо співпраці (лист приходить вам на пошту)

Темна тема, червоний акцент, анімований фон із частинками, що реагує на курсор.

## Файли

| Файл | Що це |
|---|---|
| `index.html` | Головна сторінка |
| `contact.html` | Сторінка з формою зв'язку |
| `style.css` | Усі стилі (спільні для обох сторінок) |
| `script.js` | Анімація фону, рендер/фільтр проєктів, відправка форми |
| `projects-data.js` | **Тут редагуєте список своїх проєктів** |

## Як додати свої проєкти

Відкрийте `projects-data.js` — там масив `PROJECTS`. Скопіюйте один об'єкт і заповніть свої дані:

```js
{
  title: "Назва проєкту",
  category: "Ігри",              // будь-яка назва — кнопки фільтра створяться самі
  description: "Короткий опис.",
  tags: ["Unity", "C#"],
  demo: "https://посилання-на-демо",
  repo: "https://github.com/...",
},
```

Категорії кнопок-фільтрів на головній сторінці генеруються автоматично з полів `category` — нічого більше редагувати не треба.

## Як підключити відправку форми на пошту (безкоштовно)

Сайт на GitHub Pages — статичний, без сервера, тому форма використовує безкоштовний сервіс [Formspree](https://formspree.io):

1. Зареєструйтесь на formspree.io (можна через GitHub-акаунт).
2. Створіть нову форму (**New Form**) і вкажіть пошту, на яку мають приходити повідомлення.
3. Formspree дасть посилання виду `https://formspree.io/f/xxxxxxxx` — скопіюйте його.
4. У файлі `contact.html` знайдіть рядок:
   ```html
   action="https://formspree.io/f/YOUR_FORM_ID"
   ```
   і замініть `YOUR_FORM_ID` на свій ID.
5. Formspree надішле лист-підтвердження на вашу пошту — підтвердіть її, інакше форма не запрацює.
6. Безкоштовний план дозволяє 50 повідомлень на місяць — цього зазвичай достатньо для портфоліо.

Поки ID не замінено, форма покаже про це повідомлення замість спроби відправки.

## Кольори та стиль

Усі кольори — у самому верху `style.css`, у блоці `:root`. Головний акцент:
```css
--accent: #e6394a;
```
Змініть це значення, якщо захочете інший відтінок червоного.

Анімований фон (частинки) налаштовується на початку `script.js` у секції "АНІМОВАНИЙ ФОН": можна змінити кількість частинок (`PARTICLE_COUNT`), відстань зв'язків (`LINK_DIST`) і радіус реакції на курсор (`MOUSE_DIST`).

## Як опублікувати на GitHub Pages

1. Створіть репозиторій на GitHub і зробіть його **публічним** (Settings → General → Danger Zone → Change visibility), інакше розділ Pages буде недоступний на безкоштовному акаунті.
2. Завантажте всі файли з цієї папки в репозиторій (Add file → Upload files, або через git).
3. Settings → Pages → Source: **Deploy from a branch**, гілка `main`, папка `/root` → Save.
4. Через 1–2 хвилини сайт з'явиться за посиланням, яке GitHub покаже на цій сторінці.

## Локальний перегляд

Просто відкрийте `index.html` у браузері.
